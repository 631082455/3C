一句话机制

按住技能键时，AttackInterval gate 结束的瞬间，如果输入仍按着 → 系统替玩家自动按一下。

开 / 不开

能力类型 开 理由
近战 combo 普攻（Steel） ✅ 手点累
远程段式连射普攻 ✅ 手点累
远程充能型普攻（Sparrow） ✅ 解决 gate 期间"立即重按被吞"
远程持续光束 / 弹幕喷射 ❌ 用 FSkillModuleContinuousFire，语义不同
战技 / 大招 ❌ 节奏感优先

边界规则

  • 长按 DPS = 完美手点 DPS，不提供任何额外优势
  • Combo 末段自然停止（ConditionTags 找不到下一段），不循环回 combo 1
  • 充能型不补偿被吞按键时刻，新一轮充能起点 = gate 结束时刻
  • 玩家随时松手立刻停，精度 = 一个 tick
  • 默认 off，每能力独立 opt-in；没有"全英雄统一开"的硬约束
