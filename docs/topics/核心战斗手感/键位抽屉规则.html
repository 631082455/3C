概述

一、底层原理：每个键的"人机成本"和"语义"

1.1 物理成本（手指要不要离开 WASD？）

等级 键 离手成本 适合
零成本 LMB / RMB 右手食指/中指本来就在 持续/反应性操作
零成本 Shift / Space 左小指/左拇指本来就在 持续/反应性操作
低成本 E / Q 中指/无名指抬一下，眼睛不离屏 高频主动技能
中成本 R / F 食指/无名指要拉伸 中频战术技能
高成本 C / V / X / Z 手掌要下移，Space 弃守 低频 / Team-Up
极高成本 1 / 2 / 3 / 4 整只手平移，必须低头瞄键盘 战前/低频/装备/物品

结论：传统 MOBA 的 1234 是人机成本最高的方案；RMB / Shift / E / R 让 4 个主动技能都落在零/低成本区，这是 FPS-MOBA 玩起来手感更跟手的根本原因。

1.2 语义槽位（MR 48 英雄统计规律）

MR 的设计哲学不是"技能1/2/3/4"，而是 "键就是角色"——同一个键在不同英雄上承担同样的语义。统计结果：

键 语义 典型案例 占比
LMB 主输出 / 普攻 100% 英雄都是普攻 100%
RMB 副武器 / 持续高频技能 / 按住式盾 Iron Man Unibeam、Doctor Strange Shield、Spider-Man Web-Cluster、Punisher Grenade ~90%
Shift 位移 / 自保 / 形态切换 Storm Surge、Web-Swing、Diamond Form、Indestructible Guard 100%（没有一个英雄留空 Shift）
E 战术技能 / AoE / 控制 / 增益 Lightning Realm、Pyro-Prison、Healing Bubble、Buddy Barrier ~95%
Q 大招 / 终极技 Hulk Smash、Reality Erasure、Soul Resurgence 100%
F 特殊冲击技 / 强抉择技 Karmic Revival（复活）、Liberty Rush、Moonlight Hook ~30%（备用槽）
Space 二段跳 / 飞行 / 攀爬 Incredible Leap、Hel's Descent、Skyward Leap 仅有垂直机动的英雄
C/V/X/Z Team-Up 联动技 Vibranium Bastion、Cosmic Fastball 联动专属

二、本项目"抽屉规则"

我们字母键映射 Skill 1/2/3/4 → RMB / Shift / E / R，把 MR 的语义平移过来，并针对 MOBA 特性调整。

🔴 RMB 抽屉 — 「持续/高频/瞄准型输出」

适合塞进来的技能特征： - 频次高（CD < 8s 或可连按） - 需要持续瞄准（按住式 / 多段式） - 是英雄的"第二输出手段" - 通常不是控制、不是位移

典型形态： - 蓄力射击 / 连射弹幕 - 锥形 / 直线伤害（频繁补刀用） - 按住式护盾（DocStrange 类） - 持续治疗光束（治疗英雄）

MR 案例：Iron Man Unibeam、Hela Piercing Night、Adam Cosmic Cluster、Punisher Scourge Grenade

🟡 Shift 抽屉 — 「自保 / 位移 / 形态」永远不留空

这是最重要的规则。Shift 是肌肉记忆里"出事就按"的键，没有位移的英雄也必须给 Shift 装一个"救命/转换"技能。

优先级（按从有到无依次降级）： 1. 有传统位移 → 位移（冲刺、闪现、钩拉自己、跳跃） 2. 没位移但有形态 → 形态切换（Emma Diamond Form、Wolverine Last Stand） 3. 没位移没形态但有自保 → 短无敌 / 速度爆发 / Cleanse（Hulk Indestructible Guard） 4. 上面都没有 → 短 CD 治疗自身 / 韧性增益（Venom Symbiotic Resilience）

❌ 绝对禁止放 Shift：主输出技能、需要瞄准的指向控、长前摇 AoE。 理由：玩家低血量被追时会无条件本能按 Shift；如果 Shift 是"对敌进攻技"，会死得很糊涂。

🟢 E 抽屉 — 「战术 / 控制 / AoE / 团队增益」

适合塞进来的技能特征： - CD 中等（10–20s） - 团队收益（控制、增益、群伤） - 不是救命也不是终极，是"主动开团/打节奏"用的 - 需要稍微思考时机，不是纯反应性

典型形态： - 范围控制（束缚 / 眩晕 / 缓速场） - AoE 伤害区 - 队友增益（攻速、护盾、加速） - 标记 / 减抗 / 单体硬控

MR 案例：Lightning Realm、Pyro-Prison、Healing Bubble、Buddy Barrier、Force Physics

🔵 R 抽屉 — 「大招 / 终极技」

MOBA 传统沿用，几乎没有例外。 - 长 CD（60s+） - 单局可能只放 2–3 次 - 改变战局的体量 - 视觉 / 音效最隆重

为什么 R 不放 Q（哪怕 MR 用 Q）：MOBA 玩家从 LoL/DotA 起的肌肉记忆就是 R = 大招，强行改成 Q 会让转岗玩家全员翻车。MR 用 Q 是因为它是 FPS 血统。

三、决策流程（拿到一个英雄怎么分配）

按这个顺序填，不要乱填：

Step 1：找出大招            → 放 R    （不用想）
Step 2：找出自保/位移技      → 放 Shift（找不到就找形态/无敌/治疗自己）
Step 3：剩下两个技能中，挑高频/可瞄准/输出向 → 放 RMB
Step 4：剩下那一个（战术/控制/团辅） → 放 E

冲突仲裁

  • 两个位移（一个传统位移 + 一个瞬移大招）→ 位移留 Shift，瞬移走 R
  • 既是输出又是控制（眩晕弹）→ 按 CD 长短决定：CD < 8s → RMB，CD ≥ 10s → E
  • 没有大招（罕见，纯辅助）→ R 放"最长 CD 的战略技"

四、边界问题 FAQ

Q1：高频、输出手段是不是右键好？

是。 MR 48 英雄里副武器/高频输出 90% 都在 RMB。右手食指零成本，可以一边走位一边按。规则：任何 CD < 8s 的输出向技能优先 RMB。

Q2：单体选中的技能也放右键好吗？

分情况： - "高频可重复"的单体技（持续射线、可连用钩拉、单体束缚短CD）→ RMB ✓ - "一次性单体重击 + 长 CD" → 放 E 更合适（节奏感更强，玩家会"攒着用"） - "需要选中队友放治疗" → 放 E（治疗英雄通常 RMB 已经被持续治疗光束占了）

经验法则：RMB = 频繁、Shift = 救命、E = 抉择、R = 终结。同样是单体技，频次决定塞哪儿。

Q3：Shift 如果英雄没有位移放什么好？

绝对不能空着。按优先级降级： 1. 形态切换（攻防/速度/视觉模式切换，例：钻石形态、暴怒模式） 2. 短无敌 / 短霸体 / Cleanse 解控 3. 短 CD 的"治疗自己"或"减伤护盾" 4. 速度爆发（移速 +50%/2s 之类）

底线原则：Shift 上的技能必须满足"玩家半血被追，本能按下去能活更久"。

Q4：F 键我们要不要用？

暂时不要。F 在 MR 里是"备用第 6 个槽"。我们 4 技能制如果硬塞 F 进字母键布局，会破坏 RMB/Shift/E/R 对应 1/2/3/4 的清晰映射，玩家要记两套规则。F 留给未来扩展（被动主动化、装备技、英雄专属交互）。

Q5：能不能把"治疗"放 Shift？

只对治疗自己的技能可以。给队友的治疗永远放 RMB 或 E： - 持续光束治疗 → RMB - 范围治疗 / 治疗光环 → E - 仅治疗自己的应急回复 → Shift（满足"出事就按"语义）

Q6：如果一个英雄有 2 个位移怎么办？

  • 主位移（短CD/可重复打节奏）→ Shift
  • 副位移（长CD/收尾用）→ 看用途：
  • 是终结/突进/全屏闪现 → R
  • 是战术性"过墙/绕后" → E

Q7：被动技能放在哪？

不占字母键抽屉。被动技能纯属角色机制，不参与按键映射。UI 上单独有"被动图标"显示状态即可。

五、抽屉规则一句话总结

RMB 是右手枪、Shift 是逃生绳、E 是战术匣、R 是核按钮。

拿到任何英雄都按这个语义对号入座，Shift 槽不允许留空。

附录：与数字键布局的对应关系

技能位 数字键布局 字母键布局 语义
Skill 1 1 RMB 高频输出
Skill 2 2 Shift 位移/自保
Skill 3 3 E 战术/控制
Skill 4 4 R 大招

策划填技能表时，1/2/3/4 的语义就锁定了——数字键布局是降级方案（给键鼠不熟练或习惯传统 MOBA 的玩家），但两套布局映射到同一个技能位，所以本规则同时约束两套布局下的技能设计。
