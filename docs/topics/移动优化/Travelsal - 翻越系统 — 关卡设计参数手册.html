常见问题 & 注意事项

Q：我摆了一个 1m 的平台，但角色有时翻越有时不翻，为什么？
检查角色朝向是否正对墙面（Mantle 最大 35°）；检查平台顶面是否有碰撞遮挡（Clearance 检测会查落点上方的胶囊空间）；检查是否处于 Post-traversal cooldown 冷却期（翻越结束后短暂无法再次触发）。
Q：楼梯台阶不想触发翻越，怎么设置？
背面落差 < 22cm（MantleMinStepUpDrop）的台阶系统自动跳过，不触发。确保楼梯踏步背面高差 ≤ 22cm 即可。
Q：想要强制触发 Mantle 而非 Hurdle？
加厚障碍物深度到 ≥ 59cm，或降低背面落差到 < 50cm（此时走 Step-up Mantle 路径）。
Q：高度从哪里量起？
始终以角色当前脚底为基准（不是 Actor Origin）。角色站在 1m 台上看 2m 台，目标障碍高度 = 2m − 1m = 1m，会触发 Mantle 而非 Climb。
Q：Catch 要怎么触发？
角色需要处于下落状态（跳跃后下降阶段或直接掉落），且下落速度达到 ≥ 100 cm/s（方向向下），并在 75cm 范围内检测到墙面。不需要额外输入，满足条件自动触发。
