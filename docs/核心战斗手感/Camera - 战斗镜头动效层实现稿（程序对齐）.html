1. 架构与求值管线

动效层落在现有 UGameMatchCameraComponent 内,模式栈之后的一层叠加。没动模式栈、没用 GameplayCameras 插件、无运行时新依赖。

类 / 文件角色
UCameraMotionEffect / Variant(新增 Camera/CameraMotionEffect.h/.cpp)设计师资产 CME_*/CMEV_*;IsDataValid 校验;变体经 CameraMotionEffectUtil::Resolve 展开
UGameMatchCameraComponent(改)动效栈 ActiveMotionEffects、Push/Release/ClearAll、包络求值、合成、看门狗、控制台命令、Debug 面板。旧 ACD 旁路保留
AGameMatchCameraManager(改)SetViewTarget override + OnHeroDeadGlobal 订阅 → ClearAll + StopAllCameraShakes
UHeroCameraMode_ThirdPerson(改 .cpp)每帧回写 LastPivotLocation(Orbit/臂长锚点);动效 Lag 覆盖接入(ACD lag 优先)
FSkillModuleCamera / UHeroAbility(改)模块加 MotionEffects 数组;Push/Release 桥(IsLocallyControlled 闸,EndAbility 无条件释放)
UANS_CameraMotionEffect(新增)动效挂蒙太奇窗口;状态按 MeshComp 分桶,不存 notify 对象
UTargetSelectionComponent(改 .cpp)朝向评分改读 GetPlayerViewPointForGameplay
 
GetCameraView 每帧:
1. EvaluateStack → CameraModeView(含穿墙检测,不含动效)
2. SetControlRotation ← 取值在动效前,动效旋转永不进 ControlRotation
3. 组件 transform = CameraModeView  ← 即 GameplayPOV,弹道/目标选取读它
4. UpdateMotionEffects:推进包络(Elapsed×PlayRate)、释放淡出、看门狗、到期
5. ApplyMotionEffects → 只写 DesiredView(最终 POV):
   增量求和 → 受击带软钳4°/全局20°(tanh 软拐点) → Orbit 同一 delta 四元数绕枢轴刚转
   → 臂长沿枢轴轴推拉(百分比按基准换算,最小距离保护)
   → 位移碰撞钳制:基准→最终位置单球扫(半径12,ECC_Camera),命中即收
   → FOV 求和+软钳±15
GameplayPOV 契约:动效只写 DesiredView,组件 transform 保持模式栈输出——判定纯净靠"应用点"保证,无需快照拷贝。目标选取的屏幕投影评分那半仍读渲染视图(污染小,backlog)。

2. 清理契约

事件动作
技能结束(含打断/取消/预测拒绝)EndAbility → ReleaseMotionEffectsByOwner,无条件、不看 bWasCancelled(预测拒绝以 false 到达,同路径覆盖)
模块 Remove/Pause / 蒙太奇窗口结束 / 到期按位点释放 / ANS NotifyEnd 按句柄释放(引擎三路保证送达) / 包络自然收尾
死亡 / 切观战(SetViewTarget)Manager 全清动效 + StopAllCameraShakes + 旧旁路/FOV 哨兵复位
死主看门狗每帧:无限时长条目的 OwnerObject 弱引用失效 → 强制剔除 + Error 日志(Lissandra 教训:清理不依赖健康归属者)

震屏与动效同生共死:Push 时 StartCameraShake 存弱引用,Release/ClearAll 时 Stop。观战现状:切观战清理干净,但被观战者的动效不复制给观战者(观战镜头无演绎)——体验缺失非 bug,backlog。

3. 语义定案与校验(已编码)

问题行为
重触发身份(资产, OwnerKey, SiteId)——共享库下不同技能/位点互不干扰。受击带默认 MaxAmplitudeWins
包络归属ShapeCurve(内联 FRuntimeFloatCurve,纵轴可过冲/负)有键且 Duration>0 时为唯一包络;否则梯形。Blend 字段仅中断/释放路径。PlayRate 同步缩放推进;Restart 从当前值混出
单位FOV/臂长支持 绝对|百分比,百分比按当前基准换算后求和
钳制软拐点(tanh):受击带旋转 4°、全局 20°、FOV ±15——压缩不平顶
无主推入OwnerKey=0 且 Duration<0 → 钳 1.5s + Warning(看门狗是报警器不是清洁工)

校验器(IsDataValid,存盘/验证资产时):拒绝 = Blend≤0 / ViewKick 旋转 L2>3° / Orbit>25° / 绝对FOV>±15 / 变体无父;警告 = 移动带>3s / 有曲线无有限时长 / 空动效 / 变体无覆盖。未做:引用处按操作权档位推导的预算校验(backlog)。
